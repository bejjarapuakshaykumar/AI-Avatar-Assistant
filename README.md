# 🤖 AURA – AI Avatar Assistant

AURA is an interactive AI-powered avatar assistant designed to provide real-time communication and intelligent responses using the Gemini API. The application focuses on user-friendly interaction, responsive UI design, and modern desktop web technologies.

---

# 🚀 Live Demo

🔗 https://ai-avatar-assistant-one.vercel.app/

---

# ✨ Key Features

- Integrated Gemini API for intelligent AI-generated responses
- Interactive desktop-oriented assistant interface
- Real-time query handling and response generation
- Dynamic content rendering using modern frontend technologies
- Clean and modular component-based architecture
- Optimized user interaction and usability
- Deployed using Vercel for public accessibility

---

# 🛠️ Tech Stack

Frontend:
- React / Next.js
- HTML5
- CSS3
- JavaScript

API & Services:
- Gemini API

Deployment:
- Vercel
